# Smart Hydroponic System

## Overview

This project presents a smart hydroponic system concept designed to improve plant growth through automated monitoring and control. The system focuses on soil-less farming, where plants grow in nutrient-rich water instead of traditional soil.

The concept is designed to reduce manual work, improve resource efficiency, and support controlled plant growth in urban farming, greenhouses, and other limited-space environments.

## Project Goal

The goal of this project is to create a system-level design for an automated hydroponic farming solution that monitors important plant growth conditions and adjusts them using sensors, a controller, and actuators.

## Main Features

- Automated water level monitoring
- Nutrient and pH level control
- Light intensity monitoring and adjustment
- Temperature and humidity monitoring
- Sensor-based decision-making
- User interface for system monitoring and control
- Sustainable use of water and energy
- Scalable design for different farming sizes

## System Architecture

The system is based on four main parts:

### 1. Sensors Unit

Collects real-time data from the growing environment, including:

- pH level
- Electrical conductivity
- Water level
- Light intensity
- Temperature
- Humidity

### 2. Controlling Unit

Processes sensor data and decides when adjustments are needed. The controller sends signals to actuators such as pumps, valves, lights, and fans.

### 3. Circulation Subsystem

Manages water and nutrient flow through the system. It includes pumps, valves, and nutrient delivery components.

### 4. Reservoir

Stores the water, nutrients, and balancing liquids needed for system operation.

## Functional Description

The smart hydroponic system continuously monitors the plant environment. When the system detects that a condition is outside the desired range, the controller activates the correct actuator.

Examples:

- If the water level is low, the pump adds water.
- If the pH level is not balanced, the system activates balancing valves.
- If light intensity is low, the lighting system is adjusted.
- If temperature or humidity is too high, fans can be activated.

## Testing Plan

The system design includes several testing stages:

- **Unit Testing:** Testing sensors, pumps, actuators, and control logic separately
- **Integration Testing:** Checking communication between sensors, controller, and actuators
- **User Acceptance Testing:** Evaluating usability and real-world performance
- **Performance Testing:** Testing reliability under changing conditions
- **Security Testing:** Checking data protection and authorized access
